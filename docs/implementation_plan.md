# BeautyBook v2.0 — Implementation Plan

## Текущее состояние

Проект уже имеет рабочую базу: Clerk auth, Supabase интеграция, dashboard layout, FullCalendar, CRUD для услуг/портфолио/записей, аналитику, быстрый чекаут. Задача — **переделать** существующее под новый концепт, а не писать с нуля.

**Зависит от:** 0.1, 4.1

- Endpoint отмены по телефону (`POST /api/masters/[id]/cancel`)
